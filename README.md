# UmaSwitcher

**UmaSwitcher** - A More Complicated Way To Use And Switch Between Umamusume Global and JP version. 

This utility manages the game's data directory by utilizing symbolic links within the `AppData\LocalLow\Cygames` directory, providing a technical yet efficient way to switch versions without re-downloading massive assets.
